# Open Data AI Analytics: Transport Registry 2022

## Мета проєкту
Цей проєкт присвячений аналізу відкритих даних щодо реєстрації транспортних засобів та їх власників в Україні за 2022 рік. Основна мета - вивчити особливості авторинку та підготувати структуру даних для подальшого аналізу (EDA) та застосування моделей машинного навчання.

## Джерело даних
Дані отримані з Єдиного державного вебпорталу відкритих даних:
* **Назва:** [Реєстр транспортних засобів та їх власників](https://data.gov.ua/dataset/0ffd8b75-0628-48cc-952a-9302f9799ec0)
* **Пряме покликання на архів:** [Завантажити ZIP](https://data.gov.ua/dataset/0ffd8b75-0628-48cc-952a-9302f9799ec0/resource/bef7b47b-7963-44b5-88a8-f84241137b5b/download/reestrtz2022.zip)


## Гіпотези для аналізу
1. **Гіпотеза про прогнозування типу пального:** Чи можливо передбачити тип пального автомобіля (бензин чи дизель), спираючись виключно на його базові характеристики (власну вагу, тип кузова, об'єм двигуна) та рік випуску?
2. **Залежність між двигуном та вагою":** Чи існує пряма залежність між об'ємом двигуна транспортного засобу та його вагою?
3. **Специфіка ринку реєстрацій:** Чи спостерігається на ринку реєстрацій авто в Україні переважання конкретних брендів?
